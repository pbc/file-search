# file-search

## How to run this program

This program requires Ruby 3.x which can be installed on macos via following homebrew command:

```bash
brew install ruby@3.3
```

After installing Ruby you can execute the program in the following way:

```bash
ruby ./find_content.rb <search-term> <filename>
```

E.g.

```bash
ruby ./find_content.rb something ./myfile.txt
```
